# Case-16: 朋友圈"三天可见"决策

## 元信息
- **决策者**: 张小龙、微信团队
- **决策时间**: 2017年（微信6.5.8版本）
- **决策类型**: 隐私保护功能
- **证据等级**: B级（产品历史+用户观察）
- **案例状态**: 扩展案例（精简版）

---

## 1. 决策背景

### 用户困境（2015-2017）
- **朋友圈压力**: 用户发布内容被所有好友长期可见，产生表演压力
- **社交焦虑**: 陌生人（同事/客户）通过翻看历史朋友圈了解私生活
- **手动清理痛苦**: 用户需要逐条删除历史内容（操作成本极高）

### 产品矛盾
- **价值主张**: 朋友圈是"记录生活"
- **实际使用**: 变成"社交负担"（用户开始减少发布）
- **数据下降**: 朋友圈发布频次下降（2016年开始趋势）

---

## 2. 决策逻辑

### 核心洞察：时间衰减的社交价值

**张小龙的判断**（推测）:
> "三天前的朋友圈，对今天的社交已无意义。保留历史内容增加用户负担，不如默认过期。"

**设计哲学**:
- ❌ 永久记录（传统社交网络思维）
- ✅ 阅后即焚（Snapchat启发，但微信更克制：3天而非24小时）

### 产品设计

**三天可见的实现**:
1. **默认关闭**: 需要用户手动开启（避免强制）
2. **朋友仍可见最近三天**: 保留基本社交功能
3. **自己可见全部**: 历史内容不删除，只是不对外展示
4. **可随时关闭**: 用户可以关闭恢复全部可见

**为什么是3天？**
- 1天太短（无法了解近况）
- 1周太长（仍有社交压力）
- 3天恰好：足够了解近况，又不会翻太多历史

---

## 3. 决策结果

### 短期影响（2017-2018）
- **采用率快速增长**: 大量用户开启三天可见（具体数据未公开，估计>30%）
- **争议出现**: 
  - 支持者："终于解脱了，不用担心历史被翻"
  - 反对者："显得不真诚"、"社交距离感"
- **社交礼仪讨论**: "三天可见"成为社交话题

### 长期影响（2018-2026）
- **成为主流选择**: 越来越多用户开启（尤其职场人士）
- **社交规范演变**: 
  - 从"记录生活"→"展示当下"
  - 从"长期可见"→"阅后即焚"
- **产品演化**: 
  - 2019: 增加"一个月可见"、"半年可见"选项
  - 2020: 增加"仅聊天可见"（更极致的隐私）

### 未预见的社会效应
- **求职影响**: HR无法通过朋友圈了解候选人
- **社交冷淡感**: 被批评为"中国式社交疏离"
- **代际差异**: 年轻人更倾向开启，中老年人保持全部可见

---

## 4. 决策启发式

### 启发式1: 用户负担 > 产品价值时，要做减法

**适用场景**: 功能初衷良好但用户压力大  
**决策规则**:
```
IF 功能初衷是"记录/分享"
BUT 用户实际感受是"负担/焦虑"
THEN 提供"选择退出"机制（而非强制体验）
```

**朋友圈的演进**:
- 2012: 鼓励分享（"记录生活"）
- 2017: 允许隐藏（"三天可见"）
- 2020: 提供更多选项（"仅聊天可见"）

**关键**: 不强制用户做选择，而是提供选项让用户自己平衡

### 启发式2: 克制的"阅后即焚"（3天 vs 24小时）

**决策规则**:
```
IF 借鉴竞品创新（如Snapchat的Stories）
THEN 根据自己产品定位调整参数：
  - Snapchat: 24小时（强调"此刻"）
  - 微信: 3天（平衡"此刻"与"近况"）
```

**为什么3天更合适**:
- 微信是熟人社交（需要了解近况，1天太短）
- Snapchat是娱乐社交（强调即时性，24小时够了）

### 启发式3: 默认关闭 + 用户自选（避免强制改变）

**决策规则**:
```
IF 新功能改变用户习惯
THEN 默认关闭，让用户自主选择：
  ❌ 默认开启"三天可见" → 激起用户反感
  ✅ 默认关闭，提供选项 → 用户自己决定
```

**微信的克制**:
- 不强推功能（用户可以完全不用）
- 不默认开启（避免"被改变"的反感）
- 提供多档选项（3天/1月/半年/全部）

## 5. 与其他案例对比

### vs Snapchat Stories
- **Snapchat**: 24小时后自动消失（产品强制）
- **微信**: 3天可见（用户可选）
- **差异**: Snapchat强调"即时性"，微信强调"用户控制权"

### vs Case-05 (朋友圈上线决策)
- **2012朋友圈**: 鼓励分享，定位"记录生活"
- **2017三天可见**: 保护隐私，缓解社交压力
- **演进**: 从"鼓励分享"→"尊重隐私"（产品成熟度提升）
